Colorantes Naturales




Productos de colorantes naturales



Carmín

Carmín

Se extrae de la cocion de la cochinilla, un insecto originario de América del Sur. Se utiliza principalmente en la industria alimentaria para colorear alimentos como bebidas, confitería, pasteles y productos cárnicos.

Annato

Annato

El annato es un colorante alimentario que se utiliza comúnmente para dar una tonalidad anaranjada o amarillenta a los alimentos. Se utiliza en productos como quesos, margarinas, helados y otros alimentos procesados.

Cúrcuma

Cúrcuma

Es un colorante amarillo que se obtiene a partir de las raíces secas de la planta de cúrcuma. Se utiliza en la cocina para dar sabor y color a diferentes alimentos, como arroz, productos cárnicos, salsas, etc. Se puede usar como un polvo seco o disuelto en líquido.

Caramelo

Caramelo

Se utiliza para dar color marrón oscuro a una gran variedad de alimentos y bebidas, como sodas, cervezas, salsas, caramelos y productos de panadería.

Paprika

Paprika

Es un colorante rojo-anaranjado que se obtiene a partir del pimiento rojo. Se utiliza en la cocina para agregar color y sabor a diferentes alimentos, como sopas, guisos, carnes, pollo, arroz, etc.
